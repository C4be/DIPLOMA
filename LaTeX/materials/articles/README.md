Статьи:

- [Статья](./Text_to_SQL/Text-to-SQL-emprowed%20by%20LLM.pdf) "Text-to-SQL Empowered by Large Language Models" представляет бенчмарк и систематическую оценку text-to-SQL моделей с использованием LLM.
